# Coder CN

适合中国大陆地区使用的`Coder`镜像版本。

## 🚀 快速开始

```shell
git clone https://github.com/liukunup/coder-cn.git
cd deploy

cp .env.example .env
# 修改 .env 和 nginx/nginx.conf 里面的域名等配置项

docker compose -p coder up -d
```
